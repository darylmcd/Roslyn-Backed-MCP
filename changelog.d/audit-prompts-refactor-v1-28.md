- **Changed:** Audit-oriented prompts under `ai_docs/prompts/` refactored for v1.28.0 drift and MCP-server precondition safety. (a) `deep-review-and-refactor.md` now opens with a **Phase -1 hard gate** that halts when `mcp__roslyn__server_info` is not callable — agents can no longer silently substitute `Read`/`Grep`/`Bash: dotnet build` for MCP-backed audits. Adds a `mode = full | promotion-only | read-only` parameter; `promotion-only` replaces the retired `experimental-promotion-exercise.md`. Strips hard-coded tool/prompt/skill counts and the hand-maintained skills appendix — live surface comes from `server_info`, `roslyn://server/catalog`, and `glob skills/*/SKILL.md`. Folds in the v1.19–v1.28 surface that was never audited: `workspace_warm`, `server_heartbeat`, `find_duplicated_methods`, `find_duplicate_helpers`, `find_dead_locals`, `probe_position`, `verify_pragma_suppresses`, `pragma_scope_widen`, `change_type_namespace_preview`, `trace_exception_flow`, `validate_recent_git_changes`, `replace_invocation_preview`, `preview_record_field_addition`, `record_field_add_with_satellites_preview`, `split_service_with_di_preview`, `scaffold_first_test_file_preview`, `extract_shared_expression_to_helper_preview`. Teaches Phase 6b to consume `rename_apply.MutatedSymbol` (v1.28+) for chained calls. (b) `stress-test-external-repo.md` grows a Phase -1 gate, calls `workspace_warm` in Phase 0, measures cold-vs-warm `symbol_search` in Phase 2, and adds `format_check`, `find_duplicated_methods`, `trace_exception_flow`, `validate_recent_git_changes`, `probe_position`, plus the v1.17–v1.28 preview surfaces to Phase 5. Replaces the hard-coded `C:\Code-Repo\Roslyn-Backed-MCP\…` output path with the `<Roslyn-Backed-MCP-root>/…` placeholder. (c) `experimental-promotion-exercise.md` **deleted** — ~80% redundant with deep-review; the unique scorecard-only flow is now `mode=promotion-only`. (d) `test-suite-audit.md` **deleted** — had zero `mcp__roslyn__*` references and doesn't exercise the MCP server; test-suite quality audits now live under `ai_docs/references/testing.md` guidance with Roslyn MCP tool chains (`get_complexity_metrics` / `get_cohesion_metrics` / `find_shared_members` / `find_references` scoped to `*.Tests.csproj`). `ai_docs/README.md` index + task-scoped reading guide updated; `PromptSmokeTests.cs` drift-guard comment retargeted from the retired prompt to `deep-review-and-refactor.md` Phase 16. Closes the silent-MCP-fallback failure mode users hit when agents ran these prompts without the server connected.
